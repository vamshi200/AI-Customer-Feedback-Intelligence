## 📖 Overview

Customer feedback is one of the most valuable sources of business insight, but analyzing large volumes of unstructured text manually is inefficient.

This project provides an **AI-powered dashboard** that:
- Analyzes customer feedback  
- Classifies sentiment  
- Identifies issue categories  
- Generates business insights  
- Visualizes feedback trends  

---

## 🎯 Problem Statement

Businesses struggle to:
- Analyze large volumes of feedback  
- Identify recurring issues  
- Understand customer sentiment  
- Extract actionable insights  

---

## 💡 Solution

This system:
- Processes unstructured text  
- Converts it into structured insights  
- Provides real-time analytics through an interactive UI  

---

## ✨ Features

### 📊 Dashboard

### 🧠 Feedback Analysis
